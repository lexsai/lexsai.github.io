A discord bot made during the online learning of 2020 to notify users of period changes, fetch timetabled classes, and store upcoming assignments in a public 'billboard'.

Written in Python using discord.py. Used webscraping and automation of the school's student search API to monitor period changes and fetch timetabled classes. Stored the messages of the public billboard in a postgreSQL database.

Sadly stopped working after the school started requiring authentication when requesting student timetable info. 

The discord server the bot was used on was deleted, so few records of its activity remain. :(
